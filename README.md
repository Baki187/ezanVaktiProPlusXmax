# Ezan Vakti Pro+ Xmax

Modern bir Windows system tray ezan vakti uygulaması. Bilgisayar açıldığında otomatik olarak başlar ve sistem tepsisinde (sağ alt köşe) çalışır.

## Özellikler
- COK ONEMLI İLK BAŞLATTIGINIZDA "NPM START" YAPMAYI  UNUTMAYIN YOKSA HER BİLGİSAYAR ACISILINDA 
  CALISMAZ
- 🕌 Sistem tepsisinde (system tray) çalışır — görev çubuğunda yer kaplamaz
- ⏱️ Sonraki namaza canlı geri sayım
- 📋 Tüm 6 vakit (İmsak, Güneş, Öğle, İkindi, Akşam, Yatsı)
- 🔔 Vakit girdiğinde Windows bildirimi (kapatılabilir)
- 🚀 Windows başlangıcında otomatik çalışır
- 🌙 Modern karanlık altın tema
- ⚙️ Şehir/ülke değiştirme ayarları
- 📅 Hicri takvim gösterimi

## Kurulum (Geliştirici)

```bash
npm install
npm start
```

## Kurulum Dosyası Oluşturma

```bash
npm run build
```

`dist/` klasöründe `.exe` kurulum dosyası oluşur.

## Gereksinimler

- Node.js 18+
- Windows 10/11

## API

[Aladhan API](https://aladhan.com/prayer-times-api) kullanılmaktadır. Ücretsiz, kayıt gerektirmez.
